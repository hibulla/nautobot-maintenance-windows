# Support

Use GitHub issues for public bug reports, documentation issues, and feature requests.

Before opening an issue, please check:

- the existing open issues
- the project `README.md`
- the documents in `docs/`
- the supported Nautobot and Python versions in `pyproject.toml`

For bugs, include:

- Nautobot version
- Python version
- package version or commit SHA
- installation method
- relevant traceback, API response, or job output
- steps to reproduce

For security issues, follow `SECURITY.md` instead of opening a public issue.
